객체 리터럴이란?

객체란? : 객체는 와 로 하나의 쌍으로 구성된 것

예시 : let : cat {
        name : 야옹이,
        age : 7
        }
    여기서 name 과 야옹이, age와 7을 통틀어 라고하며, name과 age는 , 야옹이와 7은 라고 칭한다.
    객체 리터럴은 이러한 프로퍼티를 직접 넣는 방법을 말한다.
